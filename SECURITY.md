# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x.x   | :white_check_mark: |
| 0.x.x   | :x:                |

## Reporting a Vulnerability

Please raise an issue on GitHub to report any security vulnerabilities in the app. If the vulnerability is potentially lethal, email us about it on support@elevate.local.
